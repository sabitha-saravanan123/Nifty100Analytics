# Nifty100 Analytics Dashboard

## Overview

Nifty100 Analytics is a financial analytics dashboard developed using Python, SQLite, and Streamlit. The project performs ETL, financial ratio analysis, stock screening, peer comparison, valuation analysis, and interactive dashboard visualization for Nifty 100 companies.

---

## Features

- ETL Pipeline
- Financial Ratio Engine
- CAGR Analysis
- Cash Flow KPIs
- Stock Screener
- Peer Comparison
- Valuation Module
- Interactive Streamlit Dashboard

---

## Technologies Used

- Python
- Pandas
- SQLite
- Streamlit
- Plotly
- OpenPyXL
- Pytest
